# SourceCery ✨

SourceCery is a single-file HTML tool designed to quickly validate the relevance of PubMed IDs (PMIDs). It provides instant context by displaying abstracts, authors, and journal details directly on mouse hover.

**Key Features:** The tool features a real-time interactive preview that recognizes multiple citation formats, such as `\cite{}`, `(PMID: ...)`, and `pmid12345`. It automatically adds author-year tags for context and includes a customizable UI with resizable panels and dark/light modes. The code was generated using Google Gemini.

**How to Use:** Simply download the `SourceCery.html` file or visit the [live version](https://chen42.github.io/SourceCery/). Open the file in a web browser and paste your text into the right panel. Hover your mouse over any detected PMID link in the preview to see details of the citation.
